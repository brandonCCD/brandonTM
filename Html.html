ESTRUCTURA BASICA HTML

Antes de ver los códigos básicos, debes saber que la primera linea de tu documento de HTML siempre debe contener la etiqueta ..DOCTYPE html> . Esto hará que el código de tu sitio se legible en cuslquier navegador.
Ahora bien, un documento HTML bien armado tiene cuatro etiquetas escenciales para que el contenido se distribuya correctamente, estas son:

  1. HTML

    2. Html está al inicio de un documento Html e indica a los navegadores que la página tiene código Html para que pueda leelo de esa forma. Sigueiendo la sintaxis del lenguaje, la etiqueta cierre html será último elemento del documento.

  2. ENCABEZADO

    3. head es la etiqueta que se utiliza para el encabezado de la página, su funcion es contener toda la información del funcionamiento del sitio. Debido a esto, es un código encriptado que las persponas que entran a la página no pueden ver.

  3. TÍTULO DE PÁGINA

    4. titlle es la etiqueta que da a un sitio un nombre para que los usuarios puedan identificarlo.
    Es el que puedes ver en las pesatañas del navegadores
